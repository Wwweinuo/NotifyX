# docker安装postgres

```shell
# 拉取 PostgreSQL 镜像
docker pull postgres:16

# 创建数据目录（Windows 可自行修改路径）
mkdir -p /docker/postgres/data

# 启动 PostgreSQL 容器
docker run -d
  --name postgres
  -e POSTGRES_USER=root
  -e POSTGRES_PASSWORD=root
  -e POSTGRES_DB=notifyx
  -p 5432:5432
  -v D:/docker/postgres/data:/var/lib/postgresql/data
  postgres:16
```



# 数据库设计

### 表 1: contacts（联系人）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| name | VARCHAR(50) | 联系人姓名 |
| relationship | VARCHAR(30) | 关系（父母/朋友/同学等） |
| birthday_type | VARCHAR(10) | 生日类型：SOLAR / LUNAR |
| birthday_date | VARCHAR(10) | 生日日期（公历 MM-DD / 农历格式） |
| lunar_month | INTEGER | 农历月份（仅农历时使用） |
| lunar_day | INTEGER | 农历日（仅农历时使用） |
| is_leap_month | BOOLEAN | 是否闰月 |
| enabled | BOOLEAN | 是否启用提醒 |
| notify_email | VARCHAR(100) | 提醒接收邮箱 |
| remark | VARCHAR(200) | 备注 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### 表 2: holidays（节日）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| name | VARCHAR(50) | 节日名称 |
| type | VARCHAR(20) | 类型：FIXED_SOLAR / FIXED_LUNAR / DYNAMIC_RULE |
| fixed_date | VARCHAR(10) | 固定日期（MM-DD） |
| lunar_month | INTEGER | 农历月份 |
| lunar_day | INTEGER | 农历日 |
| dynamic_rule | VARCHAR(100) | 动态规则描述（如 "5月第2个星期日"） |
| dynamic_month | INTEGER | 动态规则月份 |
| dynamic_week_ordinal | INTEGER | 第几个星期 |
| dynamic_day_of_week | INTEGER | 星期几（1-7） |
| enabled | BOOLEAN | 是否启用 |
| notify_email | VARCHAR(100) | 提醒接收邮箱 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### 表 3: reminder_records（提醒记录）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| reminder_type | VARCHAR(20) | 类型：BIRTHDAY / HOLIDAY |
| target_name | VARCHAR(50) | 提醒对象名称 |
| target_date | DATE | 目标日期 |
| email_to | VARCHAR(100) | 发送邮箱 |
| email_subject | VARCHAR(200) | 邮件主题 |
| email_content | TEXT | 邮件内容 |
| status | VARCHAR(20) | 状态：SUCCESS / FAILED / PENDING |
| failure_reason | VARCHAR(500) | 失败原因 |
| retry_count | INTEGER | 重试次数 |
| sent_at | TIMESTAMP | 发送时间 |
| created_at | TIMESTAMP | 创建时间 |

### 表 4: email_config（邮件配置）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| smtp_host | VARCHAR(100) | SMTP 服务器 |
| smtp_port | INTEGER | SMTP 端口 |
| username | VARCHAR(100) | 发件人邮箱 |
| password | VARCHAR(200) | 授权码（加密存储） |
| from_name | VARCHAR(50) | 发件人名称 |
| default_to_email | VARCHAR(100) | 默认接收邮箱 |
| enabled | BOOLEAN | 是否启用 |

### 表 5: scheduler_config（定时任务配置）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGSERIAL PK | 主键 |
| task_name | VARCHAR(50) | 任务名称（如 DAILY_REMINDER） |
| cron_expression | VARCHAR(50) | Cron 表达式（如 0 0 8 * * ?） |
| description | VARCHAR(200) | 任务描述 |
| enabled | BOOLEAN | 是否启用 |
| last_execution_time | TIMESTAMP | 上次执行时间 |
| next_execution_time | TIMESTAMP | 下次预计执行时间 |
| updated_at | TIMESTAMP | 更新时间 |
